# Web-Based Database Integration Interface

A comprehensive web system for registering user data and storing it in a MySQL database, featuring a clean and modern user interface.

## Technologies Used
* **Frontend**: HTML, CSS
* **Backend**: PHP
* **Database**: MySQL

##  Project Structure
* `connect.html`: Contains the data input form.
* `in.php`: Handles data reception and insertion into the database.
* `display.php`: Fetches and displays data from the database in a table.
